Notes Pane

Click Notes in the view toggle to open a side-by-side scratch pad. Drag the divider to resize. Use ◎ Snap in the Notes header to save up to 5 note checkpoints per document.

Snapshots & Sessions

A snapshot is taken automatically every 2–4 minutes if content has changed. Snapshots are also taken on tab switch or window close. Each snapshot is signed with a chained HMAC-SHA256 covering its position, timestamp, word count, and the previous snapshot's signature — making reordering, insertion, or editing detectable.

A session is a continuous period of editing. A 30-minute gap starts a new session.

Export

Document + Report — one file containing your rendered document followed by a full audit section: color-coded source showing when each passage was written, session table, and timeline.

Persistence

Content is auto-saved to your browser as a crash buffer. For durable saves, click Link session file in the status bar. The .json session file is self-contained and can restore everything on any device.
